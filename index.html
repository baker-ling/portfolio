About Me

My name is Brian Baker (he/him/his/ベーカー). I was educated as a historical linguist, and I currently work as a professional translator and interpreter of Japanese and English.

I've dabbled in programming and tech several times over my education and career, and this page is meant to collect my past work into one place for easy viewing. Programming and tech development has always immensely satisfying any time I did it, and now I hope to make it my main professional domain.

My specific interests in programming, tech, and development include the following.

  • Workplace automation for productivity improvement
  • Machine learning
  • Language technology (NLP, machine translation)
  • Internationalization and localization (specifically relating to Japanese)

Outside of linguistics and technology, I enjoy specialty coffee (esp. coffee heavy on fruity or floral notes), tea, soccer (watching and playing, though not so good at the latter), tennis (watching and playing, though pretty bad at the latter), video games. I have a modest collection of fountain pens, neckties that I almost never wear, and tenuguis that I collected while living in and travelling to Japan.

I Choose Who?

Node / Bulma / jQuery + Selectize / Handlebars / MySQL + Sequelize

This is an that recommends pokémon to use in Pokémon Go battles. The goal is to make it easier to select pokémon for particularly hard NPC battles.

You use the app by creating an account, adding the pokémon to your collection based on what you have in the Pokémon Go game, and marking pokémon you want priortized as favorites (if they are at a higher level in your actual Pokémon Go collection). Then, you can go to the Recommendations page and get recommendations based on type or enemy pokémon.

It was built as a team project with Matt Bow and Greg Riveroll.

Online Grammar Atlas of Japanese Dialects Data Viewer

Vanilla JavaScript / Google Maps API v3

This was a capstone project for my MA program in Japanese linguistics. The goal of the project was to make it easy for linguists to generate their own custom maps based on the Japanese dialect data from the Grammar Atlas of Japanese Dialects. With this tool, a linguist, dialectologist, or anyone with an interest in Japanese dialects could construct their own queries for locations to mark on the map without any special software needed, making it easier to draw insights on Japanese dialects and language development.

VBA Unicode Tools

VBA / VB Editor / Unicode

This is a pair of VBA macros for use with the VB Editor in MS Office applications on Windows. The goal of the macros is to make it easier to work with Unicode characters in VBA code. While VBA does support Unicode for string manipulation, it does not support Unicode in its own code editor. This means that if you have strings of text in a different script in your code, then those strings are coded as sequences of function calls to convert Unicode code points to characters, which is almost impossible for a human to read.

Translator Toolkit

VBA / CustomUI XML

I developed this PowerPoint add-in to make it faster and easier to do formatting work that happens before, after, or during the translation of PowerPoint files. The add-in is built with VBA to achieve the functionality and CustomUI XML to create the Ribbon customization. I localized the user interface to Japanese, and the add-in will display in Japanese if that is the user's current display language.

The next step for this project is to incorporate machine learning to remove mid-sentence line and paragraph breaks from Japanese text.

Unfortunately, the copyright to this add-in is held by TS Tech Co., Ltd., who has not granted permission to redistribute publicly.

Japanese Accent Annotator

Python / Japanese NLP tools

I developed this Python program to solve a problem that I had as a graduate teaching associate: I wanted to put accent annotations on all my Japanese example sentences that I showed my students, but the process of writing and checking those annotations was painstaking. This program uses the off-the-shelf tools fugashi (Python wrapper for morphological analysis engine Mecab), UniDic (a morphological analysis dictionary for Japanese with accent data), and ChaOne (a set of XSLTs for chunking Japanese tokens, originally meant for speech synthesis). I eventually hope to package it into a PowerPoint add-in as a gift to my former self.

Bilingual Meeting Note Taker

Materialize CSS Framework / JavaScript / LibreTranslate API / Pantry API

This is a app for taking meeting notes for a bilingual meeting where the meeting notes can be translated on the fly so that everyone in the meeting can see what notes are being taken regardless of their language. This app was motivated by an experience I had where I was interpreting for an online bilingual meeting where a Japanese-speaking participant was sharing his computer screen while typing meeting notes that English-speaking participants could not read.

The app is styled using the CSS framework Materialize. It uses the LibreTranslate API to machine translate notes at the touch of a button. It also uses the Pantry API to store past meeting notes in the cloud for later retrieval by the author or other participants.
